# 🐍 Ultra Snake v2.0

Jeu Snake ultra complet avec skins, power-ups, particules et plus !

## Télécharger le .exe

1. Va dans l'onglet **Actions** ci-dessus
2. Clique sur le dernier workflow **Build UltraSnake Windows .exe**
3. En bas de la page, télécharge **UltraSnake-Windows**
4. Extrais le ZIP → lance `UltraSnake.exe` !

## Features
- 🎨 10 Skins déblocables
- ✨ Effets de particules
- ⚡ 5 Power-ups
- 🏆 Système de combo
- 🔴 Obstacles mobiles (niveau 5+)
- 💾 Sauvegarde automatique
